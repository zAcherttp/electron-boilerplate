# Changelog

All notable changes to this template are recorded here.

## 0.1.0 — 2026-07-22

Initial template release:

- secure Electron shell with a contained production renderer and narrow IPC bridges
- React, Hono, TanStack Router and Query, Tailwind CSS, and shadcn/ui on Base UI
- adaptive custom application chrome and persisted System, Light, and Dark appearance
- structured development and packaged logging with redaction and retention
- one-command identity and icon personalization with drift detection
- formatting, linting, strict TypeScript, unit tests, and Electron smoke coverage
- cross-platform source-runtime CI and Windows NSIS packaging automation
